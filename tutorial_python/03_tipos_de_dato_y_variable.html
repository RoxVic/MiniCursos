Tipos de dato y variables

En esta clase vamos a introducir 3 tipos de datos: cadenas de caracteres, enteros y flotantes.

  • Cadenas de caracteres o string: Serán el conjunto de palabras, texto o carcateres y se van a definir siempre entre un par de comillas. Ejemplos: "Hello, word!", "abc", "2314".

  • Enteros o integer: Estan compuestos por todos los números y se definen unicamente con el carcter del número. Ejemplos: 1, 34, 789, -23.

  • Floatantes o float: Estan compuestos por todos los números con punto decimal y se definen unicamente con el carcter del número y el caracter punto que también debe de estar incluido. Ejemplos: 1.4, 34.78, 789.89, -23.5.

En Python, la instrucción para preguntar por el tipo de dato de algo es:

type()

Dentro de los paréntesis le datos por lo que queremos preguntar el tipo de dato, por ejemplo:

type("Hello, word!")

La instrucción anterior lo que hace es decirnos el tipo de dato de "Hello, word", pero recurda que si queremos que esto se muestre en pantalla debemos usar la instrucción print() como sigue:

print(type("Hello, word!"))

Variables

Para guardar un dato y reutilizarlo en nuetsro código existen las variables, que podemos imaginarlo como una cajita en donde podemos guardar un dato y le ponemos un nombre para cuando queramos usar el valor de esa cajita. PAra definar nuestra variable en Python primero vamos a poner el nombre que en este caso va a se cajita, luego el caracter "=", que va a ser el caracter para asignar un valos a esta variable y luego lo que queramos guardar, en este primer ejemplo guardaremos la cadena de caracteres "Hello, word!"

cajita = "Hello, word!"

Para usar el valor de esta variable va a ser usando su nomre, por ejemplo podriamos quere ver su contenido. Entonces primero usamos la instruccion para mostrar su contenido y dentro de los parentesis ponemos el nombre de nuestra variable:

print(cajita)

O podriamos querer ver el tipo de dato de lo que guarda la variable cajita. Primero la instrucción de mostrar en pantalla, ´¿qué queremos mostrar? lo que nos responde la instruccion type(), al indicarle que dato que esta en la vabliable cajita.

print(type(cajita))
